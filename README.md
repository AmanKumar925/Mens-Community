# Mens Community

A social community Android application where users can create posts, interact with other users, comment on discussions, and follow community members.

## Features

* User Registration & Login
* Create and Publish Blogs
* Like and Comment on Posts
* User Profiles
* Community Feed
* Follow Other Users

## Technologies Used

* Java
* Android SDK
* XML
* Firebase
